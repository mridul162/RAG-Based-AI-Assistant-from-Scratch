# Aliases
## Common names

* **English:** Honey Nuts, Honey Mixed Nuts, Honey Coated Nuts
* **Bengali:** মধুময় বাদাম (Modhumoy Badam), মধু বাদাম (Modhu Badam)

## Scientific / trade names

* **Trade Name:** Honey Mixed Nuts / Premium Honey Nuts
* **Component Ingredients (Botanical):**
* *Prunus dulcis* (Almond / কাঠবাদাম)
* *Anacardium occidentale* (Cashew nut / কাজুবাদাম)
* *Apis mellifera* Honey (Pure Natural Honey / খাঁটি মধু)
